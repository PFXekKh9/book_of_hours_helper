# Book of Hours Helper (ブック・オブ・アワーズ 補助ツール)

[Book of Hours](https://store.steampowered.com/app/1028310/BOOK_OF_HOURS/) のプレイをサポートするためのWebアプリケーションです。
ゲーム内に登場する大量の書物、記憶、アイテムの管理や、複雑なクラフトシステム（どのスキルで何が作れるか）を簡単に検索・確認することができます。
**英語Wiki等の情報をベースにしつつ、スキル名やアイテム名などを日本語化**して表示しているため、日本語でプレイしている方に最適です。

## 主な機能 (Features)

* **📕 辞典モード (Dictionary Mode)**
  * 書物、記憶、飲み物/食べ物、ツールなどのアイテムを一覧表示・検索できます。
  * 属性（心、月、蛾など）での絞り込みや、アイテム名での検索が可能です。
  * 「どの本を読めばその記憶が手に入るか」や「どうやってクラフトするか」が詳細に表示されます。

* **🛠️ 何が作れる？ (Crafting Mode)**
  * **スキルから検索:** 自分が持っている「スキル」と「レベル」を入力すると、魂カードや記憶（合計およそ+4）を足した見込みの合計属性値を自動計算し、「そのスキルで今作れる可能性のあるアイテム」をリストアップして提案します。
  * **レシピの詳細表示:** 「学者の月（レベル10）＋【必須素材: 記憶】」のように、どの属性のスロットで、どの素材を追加すれば作れるのかが直感的に分かります。
  * **属性から検索:** 欲しい属性を持つアイテムを直接検索することも可能です。

## 技術スタック (Tech Stack)

* **フロントエンド:** React, TypeScript, Vite
* **スタイリング:** Vanilla CSS (CSS Variables を用いたダークモード/グラスモーフィズムデザイン)
* **データ:** ローカルのJSONファイル (`public/data/*.json`) を使用し、Wiki等から抽出・整理した独自の日本語翻訳データセットを内包。

## セットアップ手順 (How to Run)

ローカル環境で実行する場合は、Node.js がインストールされている必要があります。

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/あなたのユーザー名/リポジトリ名.git
   ```
2. ターミナルでフロントエンドのディレクトリに移動します。
   ```bash
   cd リポジトリ名/frontend
   ```
3. 依存関係をインストールします。
   ```bash
   npm install
   ```
4. 開発用サーバーを起動します。
   ```bash
   npm run dev
   ```
5. ターミナルに表示されるURL（通常は `http://localhost:5173/` など）にブラウザでアクセスしてください。

## データについて

本アプリは内部の `frontend/public/data` 内にあるJSONデータを参照しています。
ゲームのアップデートや追加情報がある場合は、こちらのJSONを編集することでアプリ内の表示を更新できます。
